# Sustain-U
Sustain-U is a website built for college students to  report environmental issues like air, water, and waste pollution using image uploads. A machine learning model (Onnx) classifies and filters submissions, blocking invalid content before sending reports to admins for resolution, tracking, and updates with proof images.
